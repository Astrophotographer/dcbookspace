장소사용신청서 알림 연결 시안

공지사항에서 바로 들어오는 텔레그램 chat_id 등록 플로우

사용자는 이름, 휴대폰, 소속 부서, 알림 범위를 입력하고 텔레그램 봇을 시작한 뒤, 테스트 메시지까지 한 번에 확인합니다.

장소사용신청서

텔레그램/슬랙을 통한 신청 현황 실시간 알림 서비스

부서별 신청서의 진행 상태(접수, 반려, 완료)를 실시간으로 확인하고 싶으시다면, 등록하기 페이지에서 텔레그램 봇 아이디를 등록해 주세요.

본인 정보

알림 구독자 식별을 위해 이름, 휴대폰, 소속 대분류/중분류를 입력합니다.

010-1234-5678
2

알림 받을 범위

위에서 입력한 본인 소속 부서만 받을지, 모든 부서 알림을 받을지 선택합니다.

일반 사용자도 모든 부서 알림을 신청할 수 있습니다. 선택하지 않으면 본인 소속 중분류(부서)만 등록됩니다.
받을 알림
3

텔레그램 연결

추천 방식은 봇 딥링크입니다. 사용자가 Start를 누르면 n8n이 chat_id를 받아 사이트에 등록합니다.

실제 구현: 사이트가 일회용 토큰을 만들고 https://t.me/봇username?start=TOKEN 링크를 보여줍니다. n8n Telegram Trigger가 /start TOKEN을 받으면 message.chat.id를 등록 API로 보냅니다.
모바일에서는 생성 후 텔레그램 앱을 바로 열면 됩니다.
텔레그램에서 봇이 열리면 Start를 눌러 주세요. 토큰은 10분 안에 처리하는 전제로 설계합니다.
텔레그램에서 열기

자동 연결이 어려운 경우 직접 chat_id를 넣습니다. 텔레그램에서 @userinfobot 을 시작하면 본인 chat_id를 확인할 수 있습니다.
4

테스트 메시지

등록될 값을 확인하고 테스트 메시지 페이로드를 보냅니다. Webhook URL이 없으면 화면에서 성공 흐름만 시뮬레이션합니다.
